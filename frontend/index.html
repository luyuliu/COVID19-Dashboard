Welcome to the COVID-19 Dashboard!

This dashboard visualizes the spatio-temporal pattern of COVID-19 pandemic evolution for each day. The confirmed COVID-19 case statistics are shown as red proportional symbols in three scales: each country in the world (left), each state in the United States (middle), and each county in a selected state (right). Each map also has a choropleth color basemap that shows a selected relevant social-economic indicator. The line chart beneath each map demonstrates the temporal change of the confirmed case with interactivity with the proportional symbols. The scatter plot on the bottom-left corner represents county-level interrelationship between total confirmed cases and a selected socio-economic variable.

Data source: The world map is based on the 1:110M administrative data from the Natural Earth. Maps for the United States are based on the Census Bureau's 2018 Cartographic Boundary Files Shapefile at 1:20M scale. The country-level and US state-level COVID-19 confirmed case data come from the UVA COVID-19 dashboard and the JHU COVID-19 dashboard GitHub.  The county-level COVID-19 data comes from the USAFacts COVID-19 map. The COVID-19 datasets updated daily. State and county social-economic indicators are collected from American Community Survey 5-year data (2014 -2018).
